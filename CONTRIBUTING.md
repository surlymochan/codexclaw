# 贡献指南

## 开始前

- 运行 `npm install`
- 运行 `npm run init:local-guidance`
- 填写本地 `.env`

## 开发

- `npm run dev`
- `npm test`
- `npm run build`

## 规则

- 不要提交根目录的引导文件。
- 不要把密钥提交到仓库里。
- 行为变化时同步更新文档。
- 尽量保持改动小且便于审阅。

## 本地 Guidance

如果需要助手运行时文件，可以通过 `npm run init:local-guidance`
把 `docs/templates/` 里的模板复制出来。
