# Quickstart

1. Clone the repository.
2. Run `npm install`.
3. Copy `.env.example` to `.env` and fill in your Feishu and Codex settings.
4. Run `npm run init:local-guidance`.
5. Run `npm run dev` for local development, or `npm run service:install` for always-on mode.

`npm run init:local-guidance` only creates missing local files. See
[docs/local-guidance.md](docs/local-guidance.md) for details.
